# GreenPermit AI

## Features

- AI emission prediction
- Eco Score
- Risk assessment
- Sustainability recommendations
- PDF generation

## Tech Stack

React
FastAPI
Scikit-learn
Tailwind
TanStack Start

## Installation

Frontend

npm install
npm run dev

Backend

cd backend

python -m venv venv

pip install -r requirements.txt

uvicorn app.main:app --reload

Backend:
http://127.0.0.1:8000

Frontend:
http://localhost:8080
